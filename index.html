photos from anywhere

This uses the Google and Instagram APIs to geocode strings into latitude/longitude coordinates, and then search for photos taken around that location. It returns the 20 (maximum) most recent Instagram posts. You can choose a preselected landmark from the dropdown menu or search for any location (using addresses, postal codes, landmark names, etc) with the search box.

If you're going to use an address or landmark string in the search box, specify the city.
